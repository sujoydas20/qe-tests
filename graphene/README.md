# Graphene Electronic Structure (Quantum ESPRESSO)

## Objective
Study electronic properties of graphene using DFT.

## Method
- DFT (PBE functional)
- Plane-wave basis
- Quantum ESPRESSO

## Workflow
1. SCF calculation
2. NSCF calculation
3. DOS calculation
4. Band structure

## Key Parameters
- ecutwfc = 40.0
- k-points (SCF) = 9x9x1
- k-points (NSCF) = 12×12×1

## Results
- DOS plot
- Band structure plot

## Observation
- Dirac cone at K-point
- Zero band gap
